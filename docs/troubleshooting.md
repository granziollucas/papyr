# Troubleshooting

## "An error occurred. Please check the log"
Check the latest file in `logs/run_<timestamp>.log`.

## SSRN is disabled
SSRN is disabled by default. Enable it in `.env` only if you have explicit permission.

## No results
Try broader keywords, remove filters, or increase the limit.

## Downloads fail
Only official PDF URLs are downloaded. Check OA status and PDF availability.
